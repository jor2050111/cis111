# AutoPilot Express Release

**Status:** Fictional instructional scenario

## Decision

TransitTech must decide whether to delay an autonomous bus deployment after finding a rain-related braking defect.

## Evidence

* The defect appears in 0.3% of simulated heavy-rain cases.
* A smaller share could cause passenger injuries.
* Repair would delay deployment 8-12 months.
* Estimated added development and lost revenue total $9.2 million.

## Student Task

Use the evidence with the chapter's named ethical frameworks, laws,
policies, and stakeholder tests. State any assumption you add. Do not
treat a missing fact as proof.
