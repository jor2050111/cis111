# ArtForge Intellectual Property

**Status:** Fictional instructional scenario

## Decision

TechCanvas must set policy for an AI design platform trained on scraped online images.

## Evidence

* The training set contains 400 million images.
* Creators did not grant permission or receive compensation.
* Outputs can imitate a named illustrator's style.
* Customers may lack copyright protection for AI-generated logos.

## Student Task

Use the evidence with the chapter's named ethical frameworks, laws,
policies, and stakeholder tests. State any assumption you add. Do not
treat a missing fact as proof.
