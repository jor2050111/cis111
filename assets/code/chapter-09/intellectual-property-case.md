# MosaicForge IP Evidence

**Status:** Fictional instructional scenario

## Decision

MosaicForge must choose whether to keep its AI design tool live, add
controls, or pause affected uses while it closes IP evidence gaps.

## Training and Output Evidence

* Training sources include 80 million licensed images, 25 million public-domain images, and 140 million web-crawled images with no complete license record.
* The firm kept source URLs for half of the web-crawled group.
* The firm offers no creator opt-out or removal path.
* Three outputs reproduce a signature-like mark and a rare shape found in two identified artist works.
* The firm has not run a wider memorization test.
* The terms promise users "full commercial rights" to each output.

## Logo and Code Evidence

* One client used a raw output as a logo for nine months.
* The client did not search similar marks or register the logo.
* The web app includes one MIT library and one GPL library.
* The release folder has no license text or source-offer notes.
* The team has not checked how the GPL part is linked or distributed.

## Student Task

Use the facts with the chapter's IP card, fair use factors, license log,
human-work record, and training record. State any assumption you add.
Do not treat a missing fact as proof.
