# MidState Credential Incident Evidence

**Status:** Fictional instructional scenario

## Decision

MidState Community College must set the first containment, evidence,
notice, and recovery choices after stolen faculty passwords were used.

## Attack and Access Evidence

* The false message reached 200 faculty accounts.
* Thirty-two people entered passwords on the false page.
* Six accounts opened the student information system.
* The affected view holds 4,500 student records.
* Three accounts exported files. File names are known, but row counts and fields are not.
* Two accounts sent a false scholarship form that asks for bank and tax data.

## Control Evidence

* Faculty accounts do not use MFA.
* Each faculty account can open the full student view.
* Sign-in logs are kept for 30 days.
* One online backup uses the same admin account as the student system.
* The school has no tested notice map or response role chart.

## Evidence-Handling Record

* The mail team deleted the false messages before preserving a copy.
* The legal team later asked for the original message.
* No record names the admin who ran the deletion.

## Student Task

Use the facts with the incident map, identity harm map, risk register,
policy record, evidence log, response board, and notice map. State any
assumption you add. Do not treat a missing fact as proof.
