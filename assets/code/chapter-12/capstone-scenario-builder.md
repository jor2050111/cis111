# CivicLink Capstone Evidence

**Status:** Fictional instructional scenario

## Decision

Marston County must choose a full CivicLink launch, narrow pilot, or
pause.

## System and Test Evidence

* CivicLink checks forms, ranks fraud risk, and drafts benefit choices.
* A worker has 45 seconds to approve or change each draft.
* The vendor reports 92% total accuracy.
* County tests show false-flag rates of 7% for standard English web forms, 22% for translated forms, and 24% for screen-reader forms.

## Data and Notice Evidence

* Identity, health, housing, device, and location data is kept for 10 years.
* The vendor may use de-identified records, but the contract names no method or re-identification test.
* The public notice omits the AI rank, review limit, vendor use, and appeal path.
* Residents have no process to correct a bad data link.

## Worker and Security Evidence

* An engineer cited the ACM Code and asked for a narrow pilot.
* Her manager removed her from the launch meeting and told her not to write more concerns.
* A stolen vendor password opened 300 test records.
* The team reset the password but did not add MFA, keep an evidence log, or review all account actions.

## Student Task

Use the facts with the code record, critique tests, duty-to-control
record, ethical frameworks, and tools from earlier chapters. State any
assumption you add. Do not treat a missing fact as proof.
