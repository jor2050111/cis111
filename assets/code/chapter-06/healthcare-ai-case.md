# Healthcare Sepsis Prediction

**Status:** Fictional instructional scenario

## Decision

Valley Regional Medical Center is evaluating a three-year, $1.8 million AI sepsis contract.

## Evidence

* Trials reported 82% sensitivity and 95% specificity.
* The training population differs from the hospital's patients.
* Patient records would flow through vendor cloud servers.
* The proprietary model cannot receive a full independent audit.

## Student Task

Use the evidence with the chapter's named ethical frameworks, laws,
policies, and stakeholder tests. State any assumption you add. Do not
treat a missing fact as proof.
