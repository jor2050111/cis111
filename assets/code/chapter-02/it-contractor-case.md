# Database Access Request

**Status:** Fictional instructional scenario

## Decision

A contract database administrator receives an out-of-scope request for confidential customer records during an unannounced merger.

## Evidence

* The contract covers database optimization, not financial reporting.
* Company policy requires written compliance approval for exports.
* The manager has not provided that approval.
* Refusal could affect the consultant's performance review and client relationship.
